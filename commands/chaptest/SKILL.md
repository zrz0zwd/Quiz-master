# /chaptest — 章节测试

用户输入当前学习进度，agent 自动出符合出题风格的章节练习题。

## 触发条件

用户输入 `/chaptest` 或说以下话语时触发：
- "出章节测试题"
- "考我这一章"
- "按学习进度出题"

## 执行步骤

### Step 1: 确认学习参数

询问用户：
1. **科目**：热统 / 量子 / 其他
2. **章节范围**：例如 "第一章到第三章" 或 "重点考第五章"
3. **题目数量**：默认 5-8 道
4. **难度偏好**：基础巩固 / 中等提升 / 挑战难题

### Step 2: 检查风格记忆

检查是否有该科目的出题风格画像（`exam-style-{科目}`）：
- **有** → 直接使用
- **无** → 提示用户先执行 `/read-respo`

### Step 3: 对照大纲筛选考点

从大纲文件中提取指定章节的知识点，标记：
- 高频考点（必考）
- 中等考点（选考）
- 冷门考点（可能不出）

### Step 4: 生成题目

按照已学习的出题风格画像生成章节测试题。

**输出格式（Markdown 试卷模式）**：

```markdown
# {科目名称} 章节测试 — {章节范围}

> **出题风格**：模仿 {题库来源}
> **知识范围**：{章节范围}
> **题目数量**：X 道 ｜ **预计用时**：XX 分钟

---

## 一、基础题（XX 分）

**(1)**（X分）[题干...]

> [!tip] 考点
> - 对应大纲：[[章节名]]
> - 参考书：[{参考书章节} 第X题]
> - 难度：⭐⭐
```

### Step 5: 提供详解模式选项

询问用户是否需要切换到详解模式，每道题附带：
- 解题思路（审题要点、核心公式、解题步骤）
- 完整参考答案
- 评分标准
- 常见错误分析
- 拓展思考

## 输出示例

```
✅ 章节测试生成完成！

科目：热力学与统计物理
范围：第一章到第三章
题目数量：6 道
预计用时：90 分钟

难度分布：
- 基础题（⭐⭐）：3 道
- 中等题（⭐⭐⭐）：2 道
- 提高题（⭐⭐⭐⭐⭐）：1 道

已生成 Markdown 试卷，每题标注了考点来源。
需要切换到详解模式看完整解题步骤吗？
```
