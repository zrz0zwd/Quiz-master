# /anology-test — 综合模拟

依据已学习的出题风格和知识点覆盖，出跨章节综合模拟试题。

## 触发条件

用户输入 `/anology-test` 或说以下话语时触发：
- "出综合模拟卷"
- "模拟考试"
- "出一整套题"
- "按真题格式出题"

## 执行步骤

### Step 1: 确认考试参数

询问用户：
1. **科目**：热统 / 量子 / 其他
2. **考试范围**：全部 / 指定章节组合
3. **考试时长**：120min / 150min / 180min
4. **总分设置**：100分 / 150分 / 200分
5. **模拟真实考试**：严格按真题格式（含考试信息头）

### Step 2: 检查风格记忆

检查是否有该科目的出题风格画像（`exam-style-{科目}`）：
- **有** → 直接使用
- **无** → 提示用户先执行 `/read-respo`

### Step 3: 生成完整试卷

按照真题格式生成综合模拟试卷。

**输出格式（Markdown 模拟真实试卷）**：

```markdown
# {科目名称} 综合模拟试卷

> [!warning] 考试信息
> **考试时间：180分钟  满分：150分**
> 
> **范围**：第一章至第X章
> **题型参照**：{题库来源}
> **所属学科**：{学科名}

---

### 一、[大题名称]（XX分）

**(1)**（X分）[题干...]

**(2)**（X分）[题干...]

> 题型来源：{参考书章节}；{年份}真题
```

### Step 4: 附加功能

- **答案解析**：可生成完整答案附录
- **知识点地图**：标注每道题覆盖的知识点
- **薄弱点提示**：根据用户历史答题情况推荐重点复习方向

```
难度分布：

- 中等题（⭐⭐⭐）：4 道

需要生成答案附录或切换到详解模式吗？
```
